Users (Пользователи)
id (Primary Key): Уникальный идентификатор пользователя.
username: Логин пользователя.
password: паролm.
role: Роль пользователя (admin или student).


Subject (Предметы)
id (Primary Key): Уникальный идентификатор предмета.
name: Название предмета (например, ВСРПП, ПГиЗ).
total_labs: Общее количество лабораторных по предмету.


Lab (Лабораторные работы)
id (Primary Key): Уникальный идентификатор лабораторной работы.
subject_id (Foreign Key): Ссылка на предмет.
lab_number: Номер лабораторной работы.
max_points:максимальное количество баллов за лабораторную.


UserLabProgres (Прогресс по лабораторным)
id (Primary Key): Уникальный идентификатор.
user_id (Foreign Key): Ссылка на пользователя.
lab_id (Foreign Key): Ссылка на лабораторную работу.
status: Статус выполнения работы (значения: 0 - Не садился, 1 - 30%, 2 - 60%, 3 - Лаба готова, 4 - Отчет написан, 5 - Сдана).


Readiness (Общая готовность)
id (Primary Key): Уникальный идентификатор.
user_id (Foreign Key): Ссылка на пользователя.
subject_id (Foreign Key): Ссылка на предмет.
readiness: Процент готовности по предмету.
